# YourMusic

##### A free music player with no ads and customization for your needs.

## Pre-requisites

#### Get Node JS from
[https://nodejs.org/en/](https://nodejs.org/en/)

## Cloning the code
```bash
git clone https://github.com/shane-games/YourMusic.git
```

## Installing and running
```bash
npm install
```
```bash
npm start
```
